Marketing website for ARMMAX home services (painting, furniture assembly, TV mounting, handyman services, gazebo installation) built with **Next.js 14**, **TypeScript** and **Tailwind CSS**.

## Features

- Landing page with hero, services, testimonials and project gallery
- Separate pages for:
  - Interior Painting
  - Furniture Assembly
  - TV & Art Mounting
  - Handyman Services
  - Gazebo Installation
- Lead form with:
  - Name, phone, email
  - Service type and description
  - Optional file attachment
  - Telegram notification for new leads

## Tech stack

- Next.js (App Router)
- React, TypeScript
- Tailwind CSS
- Shadcn UI components
- Telegram Bot API (for new lead notifications)
